# Movie Finder
This is an console app which allows you to search for a movie, save your own rating and compare your rating with de IMDb rating. It uses the OMDb API.

# Functionality
- Search for a movie using the title
- Show the movie's information
- Save ratings in JSON
- Update your rating
- Show saved ratings
- Compare ratings with IMDb

# Tecnologies used
- Python
- Requests
- JSON
- OMDb API

#Installation
- Clone repository
- Go into folder
- Install dependencies

# How to use it
Run main.py, the menu will show you the different options.
If you want to rate a movie, you have to search it first with the option "1".
